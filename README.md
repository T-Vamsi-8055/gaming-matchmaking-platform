# 🚀 Gaming Matchmaking Platform

This platform is a full-stack application featuring a high-performance frontend and a robust backend architecture.

# 📂 Repository Structure

```
├── client/          # Vite-based React frontend
├── server/          # Node.js backend server
├── .gitignore       # Root-level Git exclusions
├── README.md        # Project Outline and Documentation
├── package-lock.json # Snapshot of the Entire Dependency Tree 
├── package.json     # Configuration Blueprint
```

# 🛠️ Tech Stack Used
- Frontend: React, Vite, ESLint, Tailwind CSS
- Backend: Node.js, Express
- Tooling: Git & GitHub
